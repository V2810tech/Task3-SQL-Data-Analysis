# SQL for Data Analysis - Task 3

## 📌 Objective
The objective of this project is to use SQL queries to extract, analyze, and summarize data from a Superstore dataset using MySQL.

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- Superstore Dataset (CSV)

## 📂 Dataset
- Sample Superstore Dataset
- Total Records: 137

## 📊 SQL Concepts Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (COUNT, SUM, AVG)
- LIMIT
- Database Creation
- Table Creation
- Data Import

## 📁 Project Files
- `SQL_Queries.sql` – Contains all SQL queries used in this project.
- `Sample-Superstore.csv.xlsx` – Dataset used for analysis.
- `Screenshots/` – Output screenshots of SQL queries.
- `README.md` – Project documentation.
